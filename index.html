🗺 Guia de Replicação — Power BI Medidas DAX e estrutura de dados
Coluna calculada: Tipo SLA
Tipo SLA = SWITCH(TRUE(), [Gerencia] = "MRO" || CONTAINSSTRING([Gerencia],"Insumos"), "Materiais", [Gerencia] = "CapEx Industrial", "CapEx", "Serviços")
Coluna calculada: SLA Normal (dias)
SLA Normal = SWITCH(TRUE(), [Tipo SLA]="Materiais" && [Spend PB MM]<0.05, 20, [Tipo SLA]="Materiais" && [Spend PB MM]<5, 30, [Tipo SLA]="Materiais", 70, [Tipo SLA]="Serviços" && [Spend PB MM]<0.12, 30, [Tipo SLA]="Serviços" && [Spend PB MM]<5, 60, [Tipo SLA]="Serviços", 120, [Tipo SLA]="CapEx" && [Spend PB MM]<5, 60, 120)
Medida: % SLA Consumido
% SLA Consumido = DIVIDE( DATEDIFF( [Data Início], TODAY(), DAY ), [SLA Normal] )
Coluna calculada: Proj. Gate 4
Proj Gate 4 = [Data Início] + SWITCH([Gate Num], 0, 75, 1, 55, 2, 35, 3, 15, 4, 0, 5, -20, 6, -60, BLANK()) // Formatar como Data
Formatação condicional: % SLA
Regra de cor (fundo): > 100% → Vermelho (#B83227) 75–100% → Laranja (#C55A11) 50–75% → Amarelo (#F0C040) < 50% → Verde (#1E7E44)

🗺 Modelo de Probabilidade — Replicação em Power BI (DAX) Todas as medidas são determinísticas e auditáveis — sem caixa preta
Prob. Base por Gate
P Base Gate = SWITCH([Gate Num], 0, 0.10, 1, 0.20, 2, 0.40, 3, 0.60, 4, 0.75, 5, 0.90, 6, 1.00, 0.15)
Multiplicador de Status
Mult Status = SWITCH([Status], "On time", 1.00, "Em negociação", 0.95, "Não iniciado", 0.85, "Risco atraso", 0.75, "Não informado", 0.70, "Atrasado", 0.50, "On hold", 0.30, "Concluído", 1.00, "Cancelado", 0.00, 0.70)
Multiplicador de SLA
Mult SLA = VAR pct = [% SLA Consumido] RETURN SWITCH(TRUE(), pct >= 1.00, 0.50, pct >= 0.90, 0.70, pct >= 0.75, 0.90, ISBLANK(pct), 0.80, 1.00)
P(Captura) Final
P Captura = MIN(1, [P Base Gate] * [Mult Status] * [Mult SLA] ) // Formatar como %
Captura Esperada (Weighted)
Captura Esperada MM = SUMX( Iniciativas, [P Captura] * [Potencial MM] )
Value at Risk
Value at Risk MM = CALCULATE( SUM([Potencial MM]), [P Captura] < 0.5, [Status] <> "Cancelado" ) // Formatar: R$ #.##0,00
RAG Score
RAG = VAR score = AVERAGEX(Iniciativas, [P Captura]) RETURN IF(score>0.55,"🟢 Verde", IF(score>0.30,"🟡 Amarelo", "🔴 Vermelho"))
